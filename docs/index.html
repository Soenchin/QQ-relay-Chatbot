Keep It Small

不堆功能、不造轮子。只做一件事：让 AI 进入 QQ 群聊。

⚡

双模路由

直调模式高速透传，智能模式上下文感知对话。两种模式按需切换。

🪶

轻量依赖

Python 3.10+ 标准库 + fastapi + websockets。无重型框架。

🧠

插件扩展

一个异步函数就是一个插件。分群开关、即时生效、无需重启。

功能概览

01

AI 双模对话

直调模式：低延迟透传；智能模式：Claude 代理 + 持续上下文。

02

主动发言

开启全量群消息权限后，机器人每 4~8 条群消息随机触发，自然接话。

03

跑团骰子

群内发送 .r d20 / .r 3d6 / .r 100，本地秒回结果。

04

WebUI 面板

群设置、管道状态、知识库、插件管理、手动发消息、实时日志。

Simple Architecture

QQ 客户端 NapCat / SnowLuma
→
relay.py 路由 + 插件 + 分发
→

三层架构，职责清晰。中间层只做一件事：把 QQ 消息送到该去的地方，再把回复带回来。

Quick Start

1

克隆仓库

2

安装依赖

pip install -r requirements.txt
3

配置

4

启动

py -3.13 -u relay.py --webui
