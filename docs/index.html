功能概览

01

双模式消息路由

支持直接透传与智能 LLM 回复两种路由模式。简单消息走高速通道，复杂对话走 AI 大脑。

02

QQ ↔ 任意平台桥接

基于 go-cqhttp / NapCat 协议适配器，无缝桥接 QQ 群、私聊与自定义后端服务。

03

热配置重载

修改配置无需重启进程。文件变动自动感知，即时生效，服务零中断。

04

极简插件接口

一个函数就是一个插件。接收消息字典，返回字符串或 None。没有继承，没有装饰器地狱。

Simple Architecture

QQ 客户端 go-cqhttp / NapCat
→
QQ Relay 核心转发层
→
后端服务 LLM / API / 自定义

三层架构，职责清晰。中间层只做一件事：把 QQ 消息送到该去的地方，再把回复带回来。

Dual Routing

DIRECT

直接路由

  • 消息透传，零延迟
  • 适合通知、Webhook、指令转发
  • 无需 LLM 调用，零 Token 消耗
  • 可配置白名单/正则匹配
SMART

智能路由

  • 上下文感知对话
  • 对接 OpenAI / DeepSeek / 本地模型
  • 自动切换多模型兜底
  • 支持系统提示词与角色设定

Quick Start

1

克隆仓库

一行命令下载源码。

git clone https://github.com/Soenchin/qq-relay-bot.git # 下载源码
2

安装依赖

仅三个轻量依赖，秒装完成。

pip install -r requirements.txt # aiohttp + websockets + pyyaml
3

配置并运行

复制一份配置模板，填写你的 QQ 和 API 信息。

cp config.example.json config.json # 复制配置
python bot.py # 启动服务
